sauvegarde est jj_mm_yyyy.tar.gz

Procédure :
Pour assurer la sauvegarde des Back-up, vous devez :

Vous connecter au système, avec les accès fournit par Inet Consulting
 Connecter le Disque Dur Externe (DDE) au serveur de sauvegarde.
 Le monter en entrant les différentes commandes suivant l’ordre
d’exécution ci-dessous :
# fdisk –l cette commande vous fournit toutes partitions/disque répertorié
par le système) le vôtre doit se trouver /dev/nom_de_votre_disque
# mkdir /mnt/mon_disque
nous créons un répertoire dans lequel on va monter une partition
# mount /dev/ nom_de_votre_disque /mnt/ mon_disque
Le support est ainsi monté dans le répertoire … /mnt/ mon_disque.
 Copie des back-up du trimestre écoulé
#cd /home/cloudbank/backup/
# cp *_mois1_* /mnt/mon_disque
#cp *_mois2_* /mnt/mon_disque
#cp *_mois3_* /mnt/mon_disque
3
 Vérification des copies
#cd /mnt/mon_disque
#ls *_mois1_* (vous avez les toutes les sauvegardes du mois1 copié sur votre support)
#ls *_mois2_*(vous avez les toutes les sauvegardes du mois2 copié sur votre support)
#ls *_mois3_*(vous avez les toutes les sauvegardes du mois3 copié sur votre support)
 Suppressions des sauvegardes copiées précédemment
Cd /home/cloudbank/backup
# rm -rf *_mois1_*
# rm -rf *_mois2_*
#rm -rf *_mois3_*

Démonter le support avec la commande
#umount /mnt/mon_disque
Pour des raisons de sécurité, il faut garder le disque dur externe hors du bâtiment abritant le
serveur.
